Prerequisites

The required software is EPICS Base >= 3.14.10, devLib2 >= 2.8, and the MSI tool (included in Base >= 3.15.1).

Base: http://www.aps.anl.gov/epics/base/R3-15/index.php

devLib2: https://github.com/epics-modules/devlib2/

MSI (Base < 3.15 only): http://www.aps.anl.gov/epics/extensions/msi/index.php
