## Quick start
run 

    ./startUI.sh -s <system name>
from this folder.

Usage: $0 [options]

Options:
- -s <system name>     The system/project name
- -g <EVG name>        Event Generator name (default: EVG0)
- -h                   This help

## Files description
- __G_EVG_VME_master.ui__ is the main screen of the EVG
